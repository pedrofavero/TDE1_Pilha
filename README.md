# TDE1_Pilha
Pré-requisitos

Java JDK 17+ instalado

Verifique:

java -version
javac -version

Compilar e executar

Cada arquivo deve conter sua própria public class com
public static void main(String[] args).

No diretório do projeto:

# compilar
javac NomeDoArquivo.java

# executar (sem .java)
java NomeDoArquivo

Exemplo:

javac Pilha.java
java Pilha
